# 🥗 Smart Diet Planner (MERN Stack)

## 📌 Project Description

Smart Diet Planner is a web-based application that provides personalized diet plans based on user health conditions and preferences. It uses modern web technologies to deliver a smooth and interactive experience.

---

## 🚀 Features

* User Authentication (Login / Signup)
* Disease-based Diet Suggestions
* AI-based Diet Recommendation
* Food Management System
* Diet Log Tracking
* Responsive Dashboard UI

---

## 🛠️ Tech Stack

* Frontend: React.js (Vite)
* Backend: Node.js + Express.js
* Database: MongoDB
* API Integration

---

## 📁 Project Structure

```
smart-diet-planner/
│
├── client/   → React Frontend
├── server/   → Node/Express Backend
```

---

## ⚙️ How to Run the Project

### 1. Clone Repository

```
git clone https://github.com/MeetThakkar10/smart-diet-planner.git
```

### 2. Install Dependencies

```
cd client
npm install

cd ../server
npm install
```

### 3. Run Project

```
# Run backend
cd server
npm start

# Run frontend
cd client
npm run dev
```
